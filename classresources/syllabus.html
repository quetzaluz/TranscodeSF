This is a syllabus for the spring 2013 Transcode class. We will update this page regularly with new links and information as the class proceeds.

If you have any questions at all, please email Naomi and Enne at info@transcodesf.org. You can also email the entire class at spring2013-discuss@transcodesf.org.

In general, homework will be assigned weekly. Each week a new pset folder will be added to the assignments git repository. (See the git guide for information on how to get it.) This problem set will be due before class on the following Tuesday. It's always easy to procrastinate, but remember that Naomi and Enne both work on Monday and Tuesdays, so if you have questions you're better off asking them early.
