Make your own version of Paint

The above is a HTML canvas and a group of divs with colors, and ids of the names of their colors. Your job is to turn it into a paint program, similar to the old mspaint you may remember. Remember, for this pset it will be best to use Chrome, since some of the APIs are different on other browsers.

All the HTML you need to get started is right here in this file, so you'll be focusing your efforts on paint.js. Over there, what you see is an init() function that is called when the HTML is done loading. It sets up the canvas to be ready to go, and creates a "context" from that canvas that you can draw on.

One thing you may find useful is the each function included in the javascript file. See if you can figure out what it does. Hint: try this in the javascript console—

      var o = {
        licorice: "black",
        eggplant: "purple",
        blueberries: "blue"
      }
      each(o, function (val, key) {
        console.log(key, val);
      });

Since we'll be using the HTML5 Canvas for this paint program, take some time to read this tutorial. Especially pay attention to the moveTo, startPath, and stroke functions, because we'll be getting some good mileage out of them.

Now that you've learned a little about the canvas, actually build a paint program. Here's the order I would suggest you'd do things in.

  • Write a function to set up the toolbar divs above to be clickable. You already have an object, COLORS with all their ids which are also the CSS code for their colors; write a loop to addEventListener for the 'click' event to each of them that sets the pen color appropriately. For now, use console.log to make sure you're getting to the click handler when you're clicking it, and setting the color to the right thing. Then call your function from init
  • Write a function to handle what happens when you press the mouse button down on the canvas (perhaps you start drawing at wherever the mouse is?). Add it as a 'mousedown' event handler on the canvas. Remember, an event handler is a function that takes an event as an argument. Maybe use console.log to record the event's offsetX and offsetY properties.
  • Write a function to handle what happens when the mouse button is released. Add it as a 'mouseup' event handler on the canvas.
  • Write a function to handle what happens when you move the mouse. This should be where you actually put pixels on the canvas. Attach it as a 'mousemove' event handler on the canvas.
After doing these things, you should have a working basic paint program! Here are some optional additions to this project:
  • A rectangle tool
  • An eraser
  • A clear button to clear the canvas
